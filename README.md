# Chessboard Interaction App

A responsive Vue 3 App that renders an interactive chessboard with click tracking.

## 🚀 Overview

This project displays a chessboard alongside a sidebar that logs user interactions. Users can click squares to highlight them and track the order of selections.

## ✨ Features

- Interactive 8x8 chessboard
- Multiple square selection (toggle on/off)
- Click history log with ordering
- Responsive layout:
  - Desktop → sidebar on the right
  - Mobile → sidebar below the board
- Board resizes to fit the viewport (minimum size enforced)
- Unit testing included for :
  - ChessBoard
  - Sidebar
  - Square

## 🛠️ Tech Stack

- Vue 3 (Composition API)
- Vite
- Vitest + Vue Test Utils

## 📦 Setup

Install dependencies:

```bash
npm install
```
